95  // Calculate BCH (Bose-Chaudhuri-Hocquenghem) code for "value" using polynomial "poly". The BCH
96  // code is used for encoding type information and version information.
97  // Example: Calculation of version information of 7.
98  // f(x) is created from 7.
99  // - 7 = 000111 in 6 bits
100  // - f(x) = x^2 + x^1 + x^0
101  // g(x) is given by the standard (p. 67)
102  // - g(x) = x^12 + x^11 + x^10 + x^9 + x^8 + x^5 + x^2 + 1
103  // Multiply f(x) by x^(18 - 6)
104  // - f'(x) = f(x) * x^(18 - 6)
105  // - f'(x) = x^14 + x^13 + x^12
106  // Calculate the remainder of f'(x) / g(x)
107  // x^2
108  // __________________________________________________
109  // g(x) )x^14 + x^13 + x^12
110  // x^14 + x^13 + x^12 + x^11 + x^10 + x^7 + x^4 + x^2
111  // --------------------------------------------------
112  // x^11 + x^10 + x^7 + x^4 + x^2
113  //
114  // The remainder is x^11 + x^10 + x^7 + x^4 + x^2
115  // Encode it in binary: 110010010100
116  // The return value is 0xc94 (1100 1001 0100)
117  //
118  // Since all coefficients in the polynomials are 1 or 0, we can do the calculation by bit
