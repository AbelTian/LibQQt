Detailed Description

The QQtNfcClient class NfcClient既可以作为Client，也可以作为Server，它是无连接的，基于数据报的协议。 在继承自QQtProtocol的xxxProtocol里，直接使用QNdefMessage就可以，不必使用QQtMessage强制转换。 这里发送和接收都是QByteArray，可以和QNdefMessage之间进行直接转换。消息没有ID，用户自行分辨Message。

这个Client是个raw客户端，数据全都在Message里的各个Record的PayLoad里， 而PayLoad里的数据格式，用户自行编订。
