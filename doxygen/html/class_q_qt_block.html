Detailed Description

The QQtBlock class QMutex，QSemphore，QCondation在gui线程会锁定gui，而我希望在gui线程中堵塞但是不要锁定gui 这个block应用场合为gui线程内部，不适用线程之间 仅仅锁定一次和解锁一次，多次锁定和解锁无用途。
