Detailed Description

QQtClickHelper 这个Helper是给Widget响应按钮事件用的 主要用于为QQtWidget系列添加按钮信号识别帮助

支持 click longClick 不支持DoubleClick, doubleClick会导致click延时

可以统计按键次数
